我的第一个 JAVA 程序

创建文件 HelloWorld.java(文件名需与类名一致), 代码如下：

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World!");
  }
}

命令行执行：

javac HelloWorld.java
java HelloWorld # output: Hello World
